# Templates Library

Reusable output formats for consistent, high-quality deliverables across all GTSE operations.

## How to Use Templates

1. **Find the right template** for your task
2. **Copy the template section** with `{{placeholders}}`
3. **Replace placeholders** with actual content
4. **Review against the example** to ensure quality
5. **Check the notes** for tips and common mistakes

## Template Standards

- **Placeholders**: Use `{{PLACEHOLDER_NAME}}` format
- **Optional sections**: Mark with `[Optional]`
- **Required fields**: No special marking (all unmarked = required)
- **Character limits**: Noted where platform restrictions apply

## Contributing New Templates

When adding templates:
1. Follow the standard format (Purpose → Template → Example → Notes)
2. Include realistic examples with GTSE products
3. Add platform-specific constraints (character limits, formatting rules)
4. Test the template with at least one real use case
