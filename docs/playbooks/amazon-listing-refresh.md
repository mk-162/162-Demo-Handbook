# Amazon Listing Refresh Playbook

Quarterly optimization to keep listings competitive and performing.

## Overview

Amazon's algorithm rewards fresh, optimized content. This quarterly refresh ensures our listings stay competitive, rank well, and convert visitors into buyers.

**Goal**: Identify and fix underperforming listings, refresh all content, maintain competitive positioning.

**Timeline**: Complete within 2 weeks of quarter start.

**Frequency**: Quarterly (Q1 = Jan, Q2 = Apr, Q3 = Jul, Q4 = Oct)

**Owner**: Marketing / Amazon Specialist

---

## Prerequisites

- [ ] Amazon Seller Central access
- [ ] Helium 10 or Jungle Scout subscription (keyword research)
- [ ] Access to product photography assets
- [ ] Google Sheets for tracking (use template below)
- [ ] Previous quarter's performance data

---

## Steps

### Step 1: Performance Review (Days 1-2)

**Identify what's working and what's not.**

1. **Pull Key Metrics** (Last 90 days)
   - Sessions (traffic)
   - Unit Session Percentage (conversion rate)
   - Buy Box percentage
   - Advertising cost of sales (ACoS)
   - Organic ranking for main keywords

2. **Categorize Listings**
   
   | Category | Criteria | Action |
   |----------|----------|--------|
   | Winners | Top 20% by revenue, CVR >15% | Protect & scale |
   | Underperformers | CVR <8%, declining sessions | Priority refresh |
   | Steady | Middle 60% | Standard refresh |
   | Zombies | <10 sessions/month | Evaluate: fix or kill |

3. **Document in Tracking Sheet**
   - ASIN, Current title, Sessions, CVR, Revenue
   - Category assignment, Priority ranking

### Step 2: Keyword Research Refresh (Days 2-4)

**Find new keyword opportunities and validate existing ones.**

1. **Export Current Keywords**
   - Backend search terms
   - Keywords in titles and bullets
   - PPC keywords with impressions

2. **Competitor Keyword Mining**
   - Use Helium 10 Cerebro on top 5 competitors per product
   - Identify gaps: Keywords they rank for that we don't
   - Note high-volume, low-competition opportunities

3. **Trend Analysis**
   - Check for seasonal keywords (especially Q4)
   - New industry terms or slang
   - Misspellings with high search volume

4. **Update Master Keyword List**
   - Primary keyword (1 per listing)
   - Secondary keywords (3-5 per listing)
   - Long-tail keywords (5-10 per listing)
   - Backend search terms (up to 250 bytes)

### Step 3: Competitor Comparison (Days 3-5)

**Know your battlefield.**

1. **Identify Top 3 Competitors** (per product category)
   - Who's in the top 3 for your main keyword?
   - Who's winning the Buy Box?

2. **Comparison Matrix**
   
   | Factor | GTSE | Comp A | Comp B | Comp C |
   |--------|------|--------|--------|--------|
   | Price | | | | |
   | Rating | | | | |
   | Reviews | | | | |
   | Main Image | | | | |
   | A+ Content | | | | |
   | Bullets | | | | |

3. **Gap Analysis**
   - Where are we weaker? (Price, reviews, images?)
   - What claims are they making that we're not?
   - What can we do better next quarter?

### Step 4: Image Audit & Refresh (Days 5-8)

**Images are 70% of the conversion equation.**

1. **Audit Current Images**
   - Main image: Pure white background? Product fills 85%+?
   - Infographics: Clear, readable, show dimensions?
   - Lifestyle: Professional, shows use case?
   - Comparison: Shows pack contents, size reference?

2. **Competitor Image Analysis**
   - Screenshot top 3 competitors' image stacks
   - What are they showing that we're not?
   - Any new infographic styles trending?

3. **Refresh Priority**
   - Underperformers: Full image refresh
   - Steady performers: 1-2 new images
   - Winners: Test new main image (A/B test if possible)

4. **Image Checklist**
   - [ ] Main image meets Amazon requirements
   - [ ] At least 5 images per listing
   - [ ] At least 1 infographic with specs
   - [ ] At least 1 lifestyle/in-use image
   - [ ] Mobile-friendly (readable at small sizes)

### Step 5: Copy Optimization (Days 6-9)

**Words sell. Make every character count.**

1. **Title Optimization**
   - Format: `[Brand] [Core Product] [Key Feature] [Size/Qty] [Color]`
   - Under 200 characters
   - Primary keyword near the front
   - No keyword stuffing

2. **Bullet Point Refresh**
   - Lead with benefit, follow with feature
   - CAPITALIZE the benefit hook
   - Include relevant specs (dimensions, strength, temp range)
   - Address common customer questions/concerns
   - Use all 5 bullets

3. **Product Description**
   - If no A+ Content: Write compelling description
   - If A+ Content: Skip (description won't display)

4. **Backend Search Terms**
   - Refresh with new keywords from Step 2
   - No repeats of words in title
   - Include misspellings, synonyms, Spanish terms
   - Stay under 250 bytes

### Step 6: A+ Content Update (Days 8-11)

**Premium content for premium conversion.**

1. **Review Current A+ Content**
   - Still accurate? (specs, claims, images)
   - Any outdated branding?
   - Comparison charts current?

2. **A+ Refresh Priorities**
   - Hero image with key benefits
   - Comparison chart vs. competitors/own products
   - Use case scenarios
   - Technical specs in visual format
   - Brand story module

3. **A+ Best Practices**
   - Mobile-first design (most Amazon traffic is mobile)
   - Large, readable text
   - Strong visual hierarchy
   - Cross-sell related products

### Step 7: Review Response Cleanup (Days 10-12)

**Show you care. Turn negatives into positives.**

1. **Audit Unanswered Reviews**
   - All 1-3 star reviews should have responses
   - Positive reviews: Thank and encourage repeat purchase

2. **Response Templates**
   
   **Negative Review Response:**
   > Thank you for your feedback. We're sorry [product] didn't meet your expectations. [Address specific issue]. Please contact us at [email] so we can make this right.
   
   **Positive Review Response:**
   > Thank you for the kind words! We're thrilled [product] is working well for you. We appreciate your business!

3. **Identify Common Complaints**
   - Document recurring issues
   - Flag for product team if quality issue
   - Update listing to set better expectations if needed

4. **Request Reviews**
   - Use Amazon's "Request a Review" button
   - Focus on recent orders (last 30 days)
   - Target satisfied customers

### Step 8: PPC Adjustment (Days 11-14)

**Align advertising with refreshed listings.**

1. **Campaign Audit**
   - Pause keywords with ACoS >50% (unless strategic)
   - Increase bids on high-converting keywords
   - Add negative keywords for irrelevant clicks

2. **New Keyword Campaigns**
   - Create campaigns for new keywords from Step 2
   - Start with auto campaigns to discover opportunities
   - Harvest converting search terms to manual campaigns

3. **Budget Reallocation**
   - Shift budget to top performers
   - Reduce spend on zombies
   - Increase for seasonal opportunities

4. **Sponsored Brand Refresh**
   - Update headlines with fresh copy
   - Rotate product selection
   - Test new creative

## Checklist

Before marking quarterly refresh complete:

- [ ] All listings categorized (Winner/Underperformer/Steady/Zombie)
- [ ] Keyword research updated for all active listings
- [ ] Competitor analysis documented
- [ ] Images audited and refreshed as needed
- [ ] Titles and bullets optimized
- [ ] A+ Content reviewed and updated
- [ ] All reviews responded to
- [ ] PPC campaigns aligned with refresh
- [ ] Tracking sheet updated with completion dates
