# Product Launch Playbook

## Overview

Complete process for launching a new product on Amazon and the BigCommerce website. This playbook covers everything from initial research through post-launch monitoring, typically spanning 2-4 weeks.

**Use this when:**
- Launching a brand new product
- Relaunching an updated product version
- Expanding an existing product to new channels

## Prerequisites

- [ ] Product samples in hand for photography
- [ ] Product specifications and technical details confirmed
- [ ] Pricing approved by management
- [ ] Inventory on hand or PO confirmed with supplier
- [ ] UPC/EAN barcode assigned
- [ ] Brand Registry approved (for Amazon A+ content)

---

## Phase 1: Research (Days 1-3)

### Step 1: Competitor Analysis
**What**: Identify top 10 competitors on Amazon for the product category

**How**:
1. Search Amazon for primary keywords
2. Document top 10 competing listings (ASIN, price, review count, rating)
3. Analyze their titles, bullet points, and images
4. Note their A+ content approach
5. Identify gaps and opportunities

**Output**: Competitor analysis spreadsheet with strengths/weaknesses

**Tools**: Helium 10, Jungle Scout, or manual research

---

### Step 2: Keyword Research
**What**: Build comprehensive keyword list for SEO and PPC

**How**:
1. Use Helium 10 Cerebro to reverse ASIN search top competitors
2. Run Magnet search for seed keywords
3. Check Amazon autocomplete suggestions
4. Compile master keyword list with search volume
5. Categorize: primary (1-3), secondary (10-15), long-tail (50+)

**Output**: Master keyword spreadsheet with priority rankings

**Tools**: Helium 10 (Cerebro, Magnet), Amazon autocomplete

---

### Step 3: Pricing Strategy
**What**: Set launch price and promotional pricing

**How**:
1. Calculate landed cost (product + shipping + duties)
2. Add Amazon fees (referral, FBA if applicable)
3. Compare to competitor pricing
4. Set target margin (minimum 30%)
5. Plan launch discount (if any)

**Output**: Pricing sheet with cost breakdown and margin analysis

**Tools**: Amazon Revenue Calculator, internal cost spreadsheet

---

## Phase 2: Content Creation (Days 4-8)

### Step 4: Title Optimization
**What**: Create keyword-rich, compelling product title

**How**:
1. Lead with brand name
2. Include primary keyword in first 80 characters
3. Add key features (size, material, quantity)
4. Stay under 200 characters
5. Follow category style guide

**Output**: Optimized title for Amazon and website

**Tools**: Helium 10 Listing Builder

---

### Step 5: Bullet Points
**What**: Write 5 benefit-focused bullet points

**How**:
1. Lead each bullet with a BENEFIT in caps
2. Follow with feature that delivers the benefit
3. Include secondary keywords naturally
4. Address common customer questions/concerns
5. Keep each bullet under 250 characters

**Output**: 5 bullet points optimized for conversion

**Tools**: Competitor research, keyword list

---

### Step 6: Product Description
**What**: Create compelling product description

**How**:
1. Write for website (full HTML/markdown)
2. Create A+ Content modules for Amazon
3. Tell the brand story
4. Highlight unique selling propositions
5. Include sizing/compatibility info

**Output**: Website description + A+ Content layout

**Tools**: Canva (for A+ graphics), brand guidelines

---

### Step 7: Product Photography
**What**: Create main image + 6 supporting images

**How**:
1. Main image: White background, product only
2. Image 2-3: Lifestyle/in-use shots
3. Image 4: Size/scale reference
4. Image 5: Key features callout
5. Image 6: What's in the box
6. Image 7: Comparison chart or infographic

**Output**: 7 optimized images (2000x2000 minimum)

**Tools**: Professional photographer or in-house setup

---

## Phase 3: BigCommerce Setup (Days 9-11)

### Step 8: Create Product in BigCommerce
**What**: Set up product listing on website

**How**:
1. Navigate to Products > Add Product
2. Enter product name, SKU, UPC
3. Set pricing (base, sale, cost)
4. Add product description (HTML)
5. Upload images with alt text
6. Set weight and dimensions
7. Configure tax settings
8. Enable inventory tracking

**Output**: Draft product listing in BigCommerce

**Tools**: BigCommerce admin panel

---

### Step 9: Category and Navigation
**What**: Assign product to correct categories

**How**:
1. Select primary category
2. Add to secondary categories if relevant
3. Ensure breadcrumb path makes sense
4. Check category SEO settings
5. Verify product appears in navigation

**Output**: Product properly categorized and findable

**Tools**: BigCommerce admin panel

---

### Step 10: Website SEO
**What**: Optimize product for Google search

**How**:
1. Set custom URL (use primary keyword)
2. Write meta title (under 60 chars)
3. Write meta description (under 155 chars)
4. Add schema markup if not automatic
5. Verify mobile appearance

**Output**: SEO-optimized product page

**Tools**: BigCommerce SEO settings, Google Search Console

---

## Phase 4: Amazon Listing (Days 12-14)

### Step 11: Create Listing in Seller Central
**What**: Build Amazon product listing

**How**:
1. Go to Catalog > Add Products
2. Select "I'm adding a product not sold on Amazon"
3. Choose correct category and subcategory
4. Enter product identifiers (UPC, brand)
5. Fill vital info (title, brand, manufacturer)
6. Add description tab content
7. Set price and quantity
8. Save and finish

**Output**: Draft Amazon listing (suppressed until complete)

**Tools**: Amazon Seller Central

---

### Step 12: Upload Images to Amazon
**What**: Add all product images

**How**:
1. Go to Catalog > Upload Images
2. Upload main image first (MAIN type)
3. Add supporting images (PT01-PT06)
4. Verify image compliance
5. Wait for processing (up to 24 hours)

**Output**: All images live on listing

**Tools**: Amazon Seller Central

---

### Step 13: Create A+ Content
**What**: Build enhanced brand content

**How**:
1. Go to Advertising > A+ Content Manager
2. Start creating Enhanced Brand Content
3. Choose modules that match your content plan
4. Upload images and text
5. Preview desktop and mobile
6. Submit for approval (allow 7 days)

**Output**: A+ Content submitted for review

**Tools**: Amazon A+ Content Manager, pre-designed graphics

---

### Step 14: Ship to FBA (if applicable)
**What**: Send inventory to Amazon fulfillment

**How**:
1. Create shipping plan in Inventory > FBA shipments
2. Set quantity and packaging type
3. Prep products per Amazon requirements
4. Print FBA labels
5. Ship to designated warehouses
6. Monitor receiving status

**Output**: Inventory received at Amazon

**Tools**: Seller Central, shipping carrier

---

## Phase 5: Launch Promotion (Days 15-21)

### Step 15: Amazon PPC Setup
**What**: Launch advertising campaigns

**How**:
1. Create Auto campaign (discovery)
2. Create Manual campaign with exact match keywords
3. Set daily budget ($20-50 for launch)
4. Set bids based on keyword relevance
5. Enable Sponsored Brands if eligible
6. Monitor daily for first week

**Output**: Active PPC campaigns driving traffic

**Tools**: Amazon Advertising Console

---

### Step 16: Email Announcement
**What**: Notify existing customer base

**How**:
1. Segment list (past purchasers, interested parties)
2. Create launch email with product images
3. Include launch discount code
4. Schedule for optimal send time
5. Set up follow-up reminder

**Output**: Launch email sent to targeted segments

**Tools**: Klaviyo or email platform

---

### Step 17: Social Media Promotion
**What**: Announce product on social channels

**How**:
1. Create product announcement posts
2. Schedule across platforms (Facebook, Instagram, LinkedIn)
3. Consider paid boost for key posts
4. Engage with comments and questions
5. Share customer reviews as they come in

**Output**: Social media launch campaign live

**Tools**: Social media scheduler, Canva

---

## Phase 6: Post-Launch Monitoring (Week 3+)

### Step 18: Monitor Metrics Daily
**What**: Track key performance indicators

**How**:
1. Check daily sales and units
2. Monitor Best Seller Rank (BSR)
3. Review PPC metrics (impressions, clicks, ACOS)
4. Watch for reviews and respond promptly
5. Track website traffic and conversion

**Output**: Daily performance log

**Tools**: Amazon Brand Analytics, Google Analytics, spreadsheet

---

### Step 19: Optimize Based on Data
**What**: Iterate and improve listing

**How**:
1. Add converting search terms to manual campaigns
2. Negative match non-converting terms
3. Adjust bids based on performance
4. A/B test main image if sales lag
5. Update copy based on customer questions

**Output**: Optimized listing and campaigns

**Tools**: Seller Central, Helium 10

---

### Step 20: Request Reviews
**What**: Build social proof

**How**:
1. Use Amazon Request a Review button (7-30 days post-delivery)
2. Send follow-up email via approved methods
3. Respond to all reviews (positive and negative)
4. Monitor for review manipulation
5. Report policy violations

**Output**: Growing review count and rating

**Tools**: Seller Central, email automation
