# Prompt: Create Blog Post

## Trigger Phrases

- "Write blog post"
- "New article"
- "Draft content"
- "Write about X"
- "Create a post on Y"

## Example Input

```
Write a blog post about why structured context is better 
than repeated prompting for AI workflows.
```

## Expected Output

Complete blog post with:
- Frontmatter (title, date, description)
- Introduction, Core Concept, Implementation, Conclusion
- Proper Markdown formatting
- Brand voice maintained

## Context Required

- [company.md](../../01-context/company.md) — Mission and philosophy
- [brand.md](../../01-context/brand.md) — Style guide

## Tools Required

- google-search — For research

## Notes

- Follow the standard blog post structure
- Use British English
- No hype words or jargon
- Save with kebab-case slug
