# Prompt: Website Content Edit

## Trigger Phrases

- "Update website"
- "Edit page"
- "Change site content"
- "Update homepage"
- "Change the headline"
- "Update navigation"

## Example Input

```
Change the homepage headline from "Stop Prompting. Start Operating." 
to "Turn AI Chaos Into Operational Excellence."
```

## Expected Output

Updated page file with:
- New content
- Preserved structure
- Brand voice maintained

## Context Required

- [brand.md](../../01-context/brand.md) — Style guide
- Target page file — Current content

## Tools Required

- local-file-system — To edit files

## Notes

- Preserve component structure
- Maintain Tailwind classes
