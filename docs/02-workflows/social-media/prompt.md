# Prompt: Social Media Distribution

## Trigger Phrases

- "Turn this article into a thread"
- "Post to social"
- "Create social content"
- "Make a LinkedIn post"
- "Tweet this"
- "Repurpose for X"

## Example Input

```
Turn this blog post into a LinkedIn post and a Twitter thread:

[Blog post content here]
```

## Expected Output

Platform-optimised content:
- LinkedIn post (150-300 characters)
- X thread (5-8 tweets)
- Suggested hashtags
- Best posting time

## Context Required

- [brand.md](../../01-context/brand.md) — Style guide
- Source content — Blog post or article

## Tools Required

- None

## Notes

- X: 280 chars per tweet, threads for depth
- LinkedIn: 1,300 max, 150-300 optimal
- Both use British English
- Include CTA in every post
