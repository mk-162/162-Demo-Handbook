# Workflow: Social Media Distribution

## Overview

This workflow repurposes long-form content into social media posts for X (Twitter) and LinkedIn.

## Prerequisites

Before starting, ensure you have:
- [ ] Source content (blog post, article, etc.)
- [ ] Target platform(s)
- [ ] Key message to convey

## Steps

### 1. Extract

Identify the key points from the source content:
- Main argument or insight
- Surprising statistic or fact
- Actionable takeaway
- Memorable quote

### 2. Format

#### For X (Twitter)

**Single Tweet:**
- 280 characters max
- Hook in first 100 characters
- One key point
- Optional: thread for longer content

**Thread Structure:**
```
Tweet 1: Hook (problem or insight)
Tweet 2-4: Context and explanation
Tweet 5-7: Supporting points
Tweet 8: Conclusion + CTA
```

#### For LinkedIn

- 1,300 characters max (optimal: 150-300)
- Professional tone
- Line breaks for readability
- Hashtags at end (3-5 max)
- Tag relevant people/companies

### 3. Adapt Voice

**X:**
- Concise, punchy
- Can be more casual
- Use threads for depth

**LinkedIn:**
- Professional but approachable
- Focus on business value
- Invite discussion

Both follow 162 brand voice:
- British English
- No jargon
- Direct and honest

### 4. Create Variants

Create 2-3 versions for testing:

| Variant | Approach |
|---------|----------|
| A | Question hook |
| B | Statistic hook |
| C | Contrarian hook |

### 5. Output

Provide:
- Platform-optimised copy
- Suggested hashtags
- Best time to post (if known)
- Link to original content
