```
- Show product
- Demonstrate key feature
- Show result

[45-60s] BENEFITS:
- Benefit 1
- Benefit 2
- Benefit 3

[60-90s] CTA:
- Where to buy
- Link in description
```

```
[0-5s] HOOK: "Quick tip for [audience]:"

[5-30s] TIP:
- State the tip clearly
- Show it in action
- Explain why it works

[30-45s] CTA:
- "Follow for more tips"
```
