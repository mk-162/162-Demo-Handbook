# Design A/B Test

## Purpose
To scientifically improve conversion rates.

## Inputs
*   `page_type`: PDP, Cart, Homepage.
*   `hypothesis`: What do we think will happen?

## Procedure
1.  **Define Metric**: Primary (Conversion Rate), Secondary (AOV).
2.  **Define Control**: Current state.
3.  **Define Variant**: The change (e.g., "Change button color to Blue").
4.  **Calculate Duration**: Use a traffic calculator to estimate how long the test needs to run.

## Output
Markdown experiment plan.
