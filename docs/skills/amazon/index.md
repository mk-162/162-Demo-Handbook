# Amazon Skills

Comprehensive skill set for Amazon marketplace management. These skills work together to optimize listings, advertising, reviews, and competitive positioning.

## Available Skills

### [Listing Audit](listing-audit/SKILL)
**Purpose:** Comprehensive audit of product listings against Amazon best practices

**Outputs:**
- Detailed audit report with scores (0-100)
- Priority fixes and quick wins
- Specific recommendations per section

**Use when:** Pre-launch, quarterly reviews, sales decline investigation

---

### [PPC Optimizer](ppc-optimizer/SKILL)
**Purpose:** Analyze and optimize Sponsored Products/Brands/Display campaigns

**Outputs:**
- ACOS analysis by campaign/ad group
- Negative keyword recommendations
- Bid adjustment suggestions
- Budget reallocation plan

**Use when:** Weekly optimization, ACOS exceeds target, scaling campaigns

---

### [Review Response](review-response/SKILL)
**Purpose:** Monitor, categorize, and respond to customer reviews

**Outputs:**
- Draft responses for all review types
- Product issue escalation reports
- Review analytics dashboard

**Use when:** Daily monitoring, negative review spikes, product quality issues

---

### [Competitor Monitor](competitor-monitor/SKILL)
**Purpose:** Track competitor pricing, reviews, launches, and strategy

**Outputs:**
- Weekly competitor digest
- Price change alerts
- New launch analysis
- Opportunity identification

**Use when:** Weekly competitive analysis, before pricing changes, strategic planning

---

## Skill Integration

These skills work best together:

```
┌─────────────────────┐
│  Competitor Monitor │ ←── Informs pricing & positioning
└──────────┬──────────┘
           ↓
┌─────────────────────┐
│   Listing Audit     │ ←── Baseline optimization
└──────────┬──────────┘
           ↓
┌─────────────────────┐
│   PPC Optimizer     │ ←── Only after listing is strong
└──────────┬──────────┘
           ↓
┌─────────────────────┐
│  Review Response    │ ←── Maintain reputation
└─────────────────────┘
```

## GTSE-Specific Guidelines

All Amazon skills should consider:
- **B2B Focus:** Tradespeople, contractors, industrial buyers
- **Product Categories:** Cable ties, safety equipment, tools
- **Brand Voice:** Professional, reliable, UK-based
- **Price Positioning:** Value (not cheapest, not premium)
