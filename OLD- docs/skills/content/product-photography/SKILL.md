# Product Photography Guidelines

## Purpose

This skill provides comprehensive guidelines for creating high-quality product photography for ecommerce listings. It covers platform-specific requirements, equipment recommendations, shooting techniques, post-processing workflows, and file management for consistent, conversion-optimizing imagery.

**Use this skill when:**
- Setting up a product photography workflow
- Training new photographers or team members
- Creating images for new product listings
- Updating existing product imagery
- Ensuring platform compliance (Amazon, eBay, website)
- Briefing external photographers or agencies

**GTSE Context:** For industrial and trade products like cable ties, fixings, and tools, photography must balance professionalism with clarity. Trade buyers need to quickly assess product specifications, while consumer buyers need lifestyle context and quality assurance.

## Inputs

- `product_name`: The product being photographed
- `product_category`: Category for style guidelines
- `target_platforms`: Where images will be used (Amazon, website, eBay, print)
- `image_count`: Number of images needed (typically 7-9)
- `special_requirements`: Any specific shots needed (scale, application, detail)
- `budget_level`: DIY, semi-pro, or professional

## Prerequisites

Before starting this skill:

1. **Equipment Available**
   - Camera or smartphone (see equipment section)
   - Lighting setup (natural or artificial)
   - Shooting surface/backdrop
   - Tripod (recommended)

2. **Products Ready**
   - Clean, undamaged samples
   - Complete packaging
   - All variants/colors available
   - Props for lifestyle shots

3. **Platform Understanding**
   - Image requirements for each platform
   - File format specifications
   - Naming conventions

4. **Software Access**
   - Photo editing software (Photoshop, Lightroom, or free alternatives)
   - Background removal tool
   - Batch processing capability

#### 4.1 Main Image (White Background)

```markdown
## White Background Shooting Process

### Setup
1. Position backdrop (paper roll or board)
2. Create gentle curve at base (no harsh lines)
3. Set up lights at 45Â° angles either side
4. Position product on sweep area
5. Mount camera on tripod

### Camera Settings
| Setting | Recommended | Why |
|---------|-------------|-----|
| Mode | Manual or Aperture Priority | Consistent exposures |
| ISO | 100-400 | Minimum noise |
| Aperture | f/8-f/11 | Maximum sharpness, good DOF |
| Shutter | Varies (1/125+ if handheld) | Freeze motion |
| White Balance | Custom or 5500K | Accurate whites |
| Format | RAW + JPEG | Editing flexibility |

### Lighting Setup (Two-Light)
```
        [Main Light 45Â°]
              â†˜
               [Product]
              â†—
        [Fill Light 45Â°]
              
        [Camera - straight on]
```

**Light Ratios:**
- Main light: Full power
- Fill light: 50-75% of main
- Results in gentle shadows, dimension

### White Background Tips
1. **Overexpose background slightly** (+0.5 to +1 EV)
2. **Keep product well-lit** but not blown out
3. **Check histogram** - spike on right (white) but no clipping
4. **Use reflectors** for shadow fill
5. **Review on computer** - camera LCD can deceive

### Common Problems & Solutions
| Problem | Solution |
|---------|----------|
| Gray background | Increase exposure or background light |
| Product too dark | Add fill light or reflector |
| Hot spots | Diffuse lights more |
| Color cast | Adjust white balance |
| Shadows too harsh | Soften lights, add fill |
| Background not pure white | Adjust in post-processing |
```

## Background Removal Methods

### Method 1: Adobe Photoshop
**Process:**
1. Open image
2. Select > Subject (AI selection)
3. Refine Edge if needed
4. Layer > Layer Mask > Reveal Selection
5. Create white layer below
6. Flatten and save

**Pro Tips:**
- Use Select and Mask workspace for fine edges
- Check at 100% zoom for edge quality
- Use Decontaminate Colors option

### Method 2: remove.bg (Online Tool)
**Best for:** Quick, simple products
**Process:**
1. Upload image to remove.bg
2. Review automatic removal
3. Edit edges if needed
4. Download with white background
**Cost:** Free (low res) or credits (~Â£0.25/image)

### Method 3: Canva Background Remover
**Process:**
1. Upload to Canva
2. Edit image > Background Remover
3. Add white background
4. Export
**Note:** Requires Canva Pro

### Method 4: Lightroom + Photoshop
**For batch processing:**
1. Basic edits in Lightroom
2. Round-trip to Photoshop for BG removal
3. Actions for repeated steps
4. Batch export

### Background Quality Check
| Check | Pass Criteria |
|-------|---------------|
| Edge quality | No fringing, halos, or rough edges |

#### 5.2 Color Correction

```markdown
## Color Correction Process

### Step 1: White Balance
**In Lightroom/Camera Raw:**
1. Use Eyedropper tool on gray card (if shot)
2. Or select "Auto" as starting point
3. Fine-tune Temp and Tint sliders
4. Target: Neutral whites, accurate colors

**Keyboard reference:**
- Too warm (yellow): Reduce Temperature
- Too cool (blue): Increase Temperature
- Too green: Increase Tint (magenta)
- Too magenta: Decrease Tint

### Step 2: Exposure Adjustment
**Goals:**
- Product well-exposed, not too bright or dark
- Whites are white (not gray or blown)
- Details visible in shadows and highlights

**Adjustments:**
- Exposure: Overall brightness
- Highlights: Recover bright areas
- Shadows: Lift dark areas
- Whites: Set white point
- Blacks: Set black point

### Step 3: Color Accuracy
**Ensure product colors match reality:**
1. Use calibrated monitor
2. Reference actual product
3. Adjust HSL/Color sliders if needed
4. Check multiple variants match packaging

### Step 4: Consistency
**All images should match:**
- Same white background brightness
- Same color temperature
- Same overall exposure feel
- Apply same preset across series

### Quick Preset Creation
**In Lightroom:**
1. Edit one image perfectly
2. Create preset from settings
3. Apply to similar images
4. Tweak individuals as needed
```

```markdown
## Product Retouching Guidelines

### What to Remove
| Remove | Keep |
|--------|------|
| Dust particles | Texture/grain |
| Scratches (unless "as new") | Normal wear (used items) |
| Fingerprints | Product design elements |
| Stray threads | Stitching |
| Sensor spots | Legitimate product details |
| Background imperfections | - |

### Retouching Tools
**In Photoshop:**
- Healing Brush: Natural corrections
- Clone Stamp: Precise copying
- Content Aware Fill: Large areas
- Patch Tool: Texture matching

### Enhancement (Careful!)
**Acceptable:**
- Slight sharpening for web
- Minor contrast boost
- Lens distortion correction
- Perspective correction

**Not Acceptable:**
- Changing product colors
- Removing actual product flaws
- Making product look different than reality
- Extreme manipulation

### Workflow Order
1. Raw conversion (exposure, white balance)
2. Background removal
3. Retouching
4. Color matching
5. Sharpening (last)
6. Export
```

### B: White Background RGB Verification

**In Photoshop:**
1. Use Eyedropper tool
2. Sample background area
3. Check Info panel: R=255, G=255, B=255
4. Anything less than 250/250/250 may appear gray

**Quick fix if gray:**
1. Select > Color Range > Highlights
2. Levels adjustment: Push whites
3. Or use Curves: Lift high end

### C: Mobile Thumbnail Test

Before uploading, view at actual thumbnail sizes:
- Amazon search: 160Ã—160 px
- Amazon PDP: 40Ã—40 px (mini)
- Website grid: 200Ã—200 px
- eBay search: 225Ã—225 px

If product unrecognizable or text unreadable at these sizes, adjust composition.

### D: Color Checker Reference

If using X-Rite/Calibrite ColorChecker:
1. Include in first shot of each lighting setup
2. In Lightroom: Select gray patch, use as white balance
3. Can create camera/lens profile for accuracy
4. Remove from final images
