## Notes

### H1 Title Format
`{{Product Name}} - {{Key Benefit or Differentiator}}`

Keep under 70 characters for SEO. Front-load the product name.

### Introduction Paragraph
- 2-3 sentences maximum
- Lead with the benefit, not the feature
- Include primary keyword naturally
- Speak to the customer's need

### Key Features
- 5-7 features maximum
- Format: **Feature Name**: Brief explanation
- Lead with the most compelling feature
- Focus on benefits, not just specs

### Specifications Table
- Include all relevant technical details
- Use consistent units (metric preferred for UK)
- Include dimensions, materials, certifications

### Use Cases
- 3-4 scenarios maximum
- Be specific about industries/applications
- Help customers self-identify

### Trust Elements
- Company credentials
- Quality certifications
- Service guarantees
- Avoid generic claims—be specific

### SEO Considerations
- Include primary keyword in H1 and first paragraph
- Use related keywords in features and specs
- Keep total word count 300-500 words
- Use proper heading hierarchy (H1 → H2 → bullets)
